oleeto

A desktop app that tracks your LeetCode grind, runs your solutions, and keeps your GitHub repo clean.

Download for macOS View Source

Log Your Solves

Add problems as you solve them. Number, title, difficulty, topic, and your solution code all tracked in one place. Paste a LeetCode link and the rest fills in automatically.

Write and Run Code

Full editor with syntax highlighting for 12 languages. Run your solution and see output right in the app. No copy-pasting to a terminal.

Auto Complexity

oleeto reads your code and detects the time and space complexity. If the guess is off, override it. Either way it shows up in your GitHub repo.

Sync to GitHub

One click pushes your solutions to a repo. Each file has a header with the problem number, difficulty, topic, link, and complexity. Sync individual problems or everything at once.

Auto-Generated README

Your repo gets a README that updates every sync. Visual activity heatmap, daily streak, problems solved, difficulty breakdown, and a full index of every solution.

Stay Sharp

oleeto surfaces problems you solved a while back so you can revisit them. Keeps your patterns fresh and your recall strong before interviews.
